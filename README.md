# STEP Face Viewer (FaceID Mode)

OpenCascade.js (WASM) と Three.js を用いて、ブラウザ上でSTEPファイルを読み込み、フェイス（面）ごとにID（FaceID）を識別してインタラクティブにカラーペイントができるWebアプリケーションです。
CADのように直感的な操作ができるよう、平行投影（オーソグラフィック）カメラや各軸方向からの視点切り替え、センタリング機能を備えています。

## 🌟 主な機能

- **STEP ファイルの動的読み込み**: ドラッグ＆ドロップまたはファイル選択ボタンから、`.step` / `.stp` 形式の3Dモデルをブラウザ内でテセレーション（メッシュ化）して表示します。
- **FaceID ベースのペイント機能**: モデルの特定のフェイスをクリック、またはドラッグすることで、境界を越えずにそのフェイス全体を直感的に塗りつぶせます。
- **CAD ライクな平行投影 (Orthographic)**: 遠近感（パース）を無効化し、寸法や形状を正確に把握しやすい描画モードを採用しています。
- **ワンクリック視点切り替え**: 各座標軸の正負方向（`+X`, `-X`, `+Y`, `-Y`, `+Z`, `-Z`）から、現在のズーム距離を維持したまま瞬時に視点を回り込ませることができます。
- **スマート・センタリング**: カメラの現在の画角や向き（傾き）を完全に維持したまま、オブジェクトの幾何学中心へ視点を平行移動します。
- **エッジ表示切替**: 3Dモデルの輪郭線（エッジ）の表示/非表示を自由に切り替えられます。
- **履歴管理 (Undo)**: 直近20回までのペイント操作を元に戻すことができます。
- **カラー状態の保存・読み込み**: 適用したカラーデータを JSON 形式でエクスポートし、同じSTEPファイルに対して後から再読込して復元することが可能です。

## 📂 構成ファイル

- **`index.html`**: アプリケーションのUIレイアウトおよびスタイル（サイドバー、カラーパレット、操作ボタン、情報ボックスなど）を定義しています。
- **`main.js`**: WebGL (Three.js) による描画制御、OpenCascade.js を用いたSTEP解析、カメラ制御、およびペイントロジックの実装コアです。
- **`coi-serviceworker.js`**: GitHub Pages などのホスティング環境で HTTP ヘッダーを制御できない場合でも、ServiceWorker 経由で `COOP` / `COEP` ヘッダーを注入し、WASM マルチスレッド版に必要な `SharedArrayBuffer` を有効化します。
